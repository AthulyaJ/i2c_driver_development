# I2C Driver Development (Bare-Metal)

## Description
This project implements a register-level I2C driver for microcontrollers, supporting master transmit and receive operations.

## Features
- Start and Stop condition generation
- Slave addressing (7-bit)
- ACK/NACK handling
- Master transmit and receive

## Concepts Covered
- Memory-mapped registers
- Peripheral configuration
- I2C protocol communication

## Learning Outcome
Gained hands-on experience in low-level driver development and communication between MCU and external devices.

## Future Improvements
- Interrupt-based I2C handling
- Slave mode implementation
